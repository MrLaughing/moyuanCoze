# 植物素材目录

此目录存放所有植物的 PNG 素材图。

## 命名规范
`{plantId}_lv{等级}_w{枯萎阶段}.png`

- plantId: changpu, wenzhu, orchid, bajiao, shuixian, mozhu, yelaixiang, guihua, yehehua, lamei, tanhua, yuejiancao, pine, cypress, plum, bamboo, ginkgo, guteng, chrysanthemum, ziteng, lotus, haitang, peony, bodhi, lingzhi, bianhua, bingtilian
- 等级: lv1 ~ lv5
- 枯萎阶段: none, fade, wither, severe, dead

## 每种植物需要9张图
- lv1_none, lv2_none, lv3_none, lv4_none, lv5_none (5张正常状态)
- lv3_fade, lv3_severe, lv3_dead (3张枯萎示例，使用lv3为基础)

总计: 27种 × 9张 = 243张 PNG
